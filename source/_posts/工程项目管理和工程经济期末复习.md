# 工程项目管理和工程经济期末复习



1. NPV（**Net Present Value**）净现值

2. PC（**Present Cost**）现值成本（也常译作：费用现值）

3. NAV（**Net Annual Value**）净年值

4. AC（**Annual Cost** ）年成本



时间相同 → 把钱统一折算到现在 → 比 NPV / PC
时间不同 → 把钱统一折算成每年 → 比 NAV / AC



| 系数        | 原来的钱           | 换算成什么 | 常用于               |
| ----------- | ------------------ | ---------- | -------------------- |
| `(P/A,i,n)` | 每年等额的钱 A     | 现在值 P   | 每年收益、每年成本   |
| `(P/F,i,n)` | 未来某一年一笔钱 F | 现在值 P   | 残值、期末一次性收入 |

| 符号        | 把什么变成每年 A        | 常用于   |
| ----------- | ----------------------- | -------- |
| `(A/P,i,n)` | 现在的一笔钱 P → 每年 A | 初始投资 |
| `(A/F,i,n)` | 未来的一笔钱 F → 每年 A | 期末残值 |

| 系数  | 方向  | 人话                 |
| ----- | ----- | -------------------- |
| `P/A` | A → P | 每年一笔钱，折到现在 |
| `P/F` | F → P | 未来一笔钱，折到现在 |
| `A/P` | P → A | 现在一笔钱，摊到每年 |
| `A/F` | F → A | 未来一笔钱，摊到每年 |



| 情况               | 思路                     | 指标     |
| ------------------ | ------------------------ | -------- |
| 有收益，时间相同   | 折到现在，看谁赚得多     | NPV 最大 |
| 只有成本，时间相同 | 折到现在，看谁花得少     | PC 最小  |
| 有收益，时间不同   | 折成年值，看每年谁赚得多 | NAV 最大 |
| 只有成本，时间不同 | 折成年值，看每年谁花得少 | AC 最小  |

## 题目 1：收益型项目综合题

某企业准备投资建设一个项目，基准收益率为 10%。

项目资料如下：

```text
初始投资：120 万元
寿命期：5 年
每年营业收入：55 万元
每年经营成本：20 万元
第 5 年末残值：10 万元
```

已知：

```text
(P/A,10%,5) = 3.7908
(P/F,10%,5) = 0.6209
```

请完成：

### 1）计算每年净收益

提示：

```text
年净收益 = 年营业收入 - 年经营成本
```

------

### 2）用净现值 NPV 判断该项目是否可行

提示：

```text
NPV = 年净收益 × (P/A,i,n) + 残值 × (P/F,i,n) - 初始投资
```

判断规则：

```text
NPV > 0，可行
NPV < 0，不可行
```

------

### 3）计算静态投资回收期

提示：

```text
静态投资回收期 = 初始投资 / 年净收益
```

------

### 4）另有一个项目 B，资料如下：

```text
初始投资：100 万元
寿命期：5 年
每年净收益：30 万元
无残值
```

已知：

```text
当 i = 14% 时，(P/A,14%,5) = 3.4331
当 i = 16% 时，(P/A,16%,5) = 3.2743
```

请用插值法求项目 B 的 IRR。

提示：

```text
NPV = 年净收益 × (P/A,i,n) - 初始投资
```

先算：

```text
NPV_14%
NPV_16%
```

再用：

```text
IRR = i1 + NPV1 / (NPV1 + |NPV2|) × (i2 - i1)
```

## 题目 2：方案比选综合题

某工程需要选择设备方案，基准收益率为 10%。

------

### 第一问：寿命期相同，只有成本

两个设备方案都能完成同样功能，寿命期均为 5 年。

| 方案 | 初始投资 | 年运行成本 | 第 5 年末残值 |
| ---- | -------- | ---------- | ------------- |
| A    | 100 万   | 20 万/年   | 10 万         |
| B    | 130 万   | 12 万/年   | 15 万         |

已知：

```text
(P/A,10%,5) = 3.7908
(P/F,10%,5) = 0.6209
```

请用费用现值 PC 选择较优方案。

提示：

```text
PC = 初始投资 + 年运行成本 × (P/A,i,n) - 残值 × (P/F,i,n)
```

判断规则：

```text
PC 越小越好
```

------

### 第二问：寿命期不同，只有成本

另有两个设备方案，寿命期不同。

| 方案 | 初始投资 | 年运行成本 | 残值  | 寿命期 |
| ---- | -------- | ---------- | ----- | ------ |
| C    | 100 万   | 20 万/年   | 10 万 | 5 年   |
| D    | 150 万   | 12 万/年   | 20 万 | 10 年  |

已知：

```text
(A/P,10%,5) = 0.2638
(A/F,10%,5) = 0.1638

(A/P,10%,10) = 0.1627
(A/F,10%,10) = 0.0627
```

请用年费用 AC 选择较优方案。

提示：

```text
AC = 初始投资 × (A/P,i,n) + 年运行成本 - 残值 × (A/F,i,n)
```

判断规则：

```text
AC 越小越好
```

------

### 第三问：寿命期不同，有收益

两个投资方案资料如下：

| 方案 | 初始投资 | 年净收益 | 残值  | 寿命期 |
| ---- | -------- | -------- | ----- | ------ |
| E    | 100 万   | 35 万/年 | 10 万 | 5 年   |
| F    | 150 万   | 45 万/年 | 20 万 | 10 年  |

仍使用：

```text
(A/P,10%,5) = 0.2638
(A/F,10%,5) = 0.1638

(A/P,10%,10) = 0.1627
(A/F,10%,10) = 0.0627
```

请用净年值 NAV 选择较优方案。

提示：

```text
NAV = 年净收益 - 初始投资 × (A/P,i,n) + 残值 × (A/F,i,n)
```

判断规则：

```text
NAV 越大越好
```
