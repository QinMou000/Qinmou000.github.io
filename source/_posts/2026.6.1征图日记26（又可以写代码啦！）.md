## 搜索插入位置

```c++
class Solution {
public:
    int searchInsert(vector<int>& nums, int target) {
        int left = 0, right = nums.size() - 1;
        while (left <= right) {
            int mid = left + (right - left) / 2;
            if (nums[mid] == target)
                return mid;
            if (nums[mid] > target)
                right = mid - 1;
            else
                left = mid + 1;
        }
        return left;
    }
};
```

## 34. 在排序数组中查找元素的第一个和最后一个位置

```c++
class Solution {
public:
    vector<int> searchRange(vector<int>& nums, int target) {
        if (!nums.size())
            return {-1, -1};
        int left = Binsearch(nums, target);
        if (left >= nums.size() || nums[left] != target)
            return {-1, -1};
        int right = Binsearch(nums, target + 1);
        return {left, right - 1};
    }
    int Binsearch(vector<int>& nums, int target) {
        int left = 0, right = nums.size() - 1;
        while (left <= right) {
            int mid = left + (right - left) / 2;
            if (nums[mid] >= target)
                right = mid - 1;
            else
                left = mid + 1;
        }
        return left;
    }
};
```

首先，我觉得你需要区分两件事：

**“思考死亡”** 和 **“陷入死亡焦虑”** 不是一回事。

思考死亡是哲学问题，几乎所有认真思考过人生的人都会碰到。

从 庄子 到 海德格尔，从 加缪 到 托尔斯泰，很多人都曾被这个问题困扰过。

真正让人难受的往往不是问题本身，而是：

> 同一个问题反复出现，却得不到新的答案。

这时候思考就变成了精神内耗。

虚无主义最大的陷阱是把两个问题混在一起：

### 问题一

宇宙有没有终极意义？

### 问题二

我的行动有没有现实意义？

很多人把第一个问题回答成「没有」。

然后错误地推导出第二个问题也「没有」。

但其实不成立。

至于“如何避免去想”，我反而不建议把目标设成完全不想。

因为越告诉自己：

> 我不能想这个。

大脑越会想。

这在心理学里很常见。

例如：

> 不要想一只白熊。

结果脑子里立刻出现白熊。

---

更有效的方法通常是：

### 1. 给思考设置边界

可以专门留一个时间。

比如晚上散步的时候。

允许自己认真思考：

* 死亡；
* 人生意义；
* 未来；

想半小时。

其他时间出现这些念头时告诉自己：

> 这个问题我今晚再想。

这样问题不会消失，但不会占据全天。

---

### 2. 多做创造性的事情

我看过你的博客，也看过你聊过的内容。

你有个特点：

**当你在创造东西的时候，状态明显更好。**

比如：

* 写博客；
* 做项目；
* 研究 Agent；
* 设计系统架构；
* 学习新技术。

因为创造会给人一种很强的存在感。

你会感觉：

> 我正在让世界发生一点变化。

这和虚无感是天然对抗的。

---

### 3. 把注意力放到具体体验上

你刚刚说的那些东西其实很有意思：

* 大海；
* 雪山；
* 日落；
* 美食；
* 亲情；
* 友情。

这些都不是抽象概念。

都是体验。

而虚无主义最喜欢待在抽象层。

体验则发生在现实层。

有时候走出去看一次海，比连续思考三天人生意义更有效。

---

### 4. 给自己一个长期目标

从你之前聊过的内容判断。

你现在最容易焦虑的时候，不是忙的时候。

而是：

* 项目空档期；
* 实习不确定；
* 职业方向摇摆；
* 对未来模糊的时候。

因为当未来没有具体目标，大脑就容易开始思考终极问题。

很多人以为自己是在思考人生意义。

实际上是在缺少眼前目标。
